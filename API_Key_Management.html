Sample Integration Use Cases (FAQs)

  • How can I sync employee master data with my CRM or L&D system?
    Use: Get Employee Information (v1) API
    Purpose: Fetch basic employee details for a given period.
  • How do I automate new joiner creation from my global HR system?
    Use: Add Employee API
    Purpose: Create employee records in HROne directly from an external HRMS or ATS.
  • How can I integrate employee leave data into my ERP or attendance system?
    Use: Get Leave Balance Report API
    Purpose: Retrieve real-time leave balances for employees.
  • How do I mark attendance when an employee logs in through CRM?

    Purpose: Automatically mark attendance in HROne from a CRM login event.

How to Access APIs

  • Login to HROne
  • Navigate: Setup → Integration → API Key Management
  • Click Create API Key
  • Select available scopes you want to consume
Please note there is a one-time enablement fee of ₹ 15,000 to access the portal. You can access all of the available APIs and generate as many keys as needed. With this you get 4 hours of technical support, if needed. The API Key Management tool inside HROne makes it simple to generate keys and assign scopes. Different keys can be generated for different scopes, different API, for different partners.

Things to Know

  • Generate different API keys for different scopes/partners
  • Whitelist IPs per API key for secure access
  • API rate limit: 100 requests per minute per scope
  • Logs accessible within product for monitoring
Security and governance are built-in. You can assign granular scopes, restrict usage by IP, and track API consumption logs directly inside the product.
